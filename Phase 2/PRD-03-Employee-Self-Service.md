# PRD-03 — Employee Self-service (Mobile-first Portal) — HRM-lite

## 1) Mục tiêu
- Nhân viên tự xem lịch/công/phép/payslip
- Nhân viên tự tạo request
- Manager duyệt nhanh trên mobile

---

## 2) IA Mobile (Bottom nav)
1) Trang chủ
2) Lịch làm
3) Chấm công
4) Yêu cầu
5) Cá nhân

---

## 3) Chức năng
- Trang chủ: ca hôm nay, check-in/out, thông báo
- Lịch làm: week view + detail
- Chấm công: check-in/out + gps status
- Yêu cầu: tạo request + list + timeline
- Cá nhân: hồ sơ, payslip, phép, device status

---

## 4) UI requirements
- PWA style, responsive mobile-first
- Load nhanh < 2s
- Theme dribbble glass cards
- Label tiếng Việt đơn giản

---

## 5) Acceptance
- 80% request tạo từ portal
- Manager duyệt < 30s
